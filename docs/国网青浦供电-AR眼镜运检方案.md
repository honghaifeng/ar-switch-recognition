# 国网青浦供电公司运检部门 AR智能眼镜辅助方案

> 编制日期：2026年3月

---

## 一、现状痛点分析

### 1.1 开关设备型号繁杂

国网运检部门管辖的开关设备种类极多，运检人员需要记忆和区分大量型号：

| 类别 | 常见型号 | 电压等级 | 说明 |
|------|---------|---------|------|
| **真空断路器** | ZN63 (VS1) 系列 | 10kV户内 | 最广泛，固定式/手车式 |
| **真空断路器** | 柱上真空断路器 | 10kV户外 | 配网柱上，带故障检测 |
| **SF6断路器** | LW36-126 系列 | 110kV及以上 | 户外高压，输电网控制 |
| **隔离开关** | GW4（水平双柱旋转式） | 35kV+ | 全国在运约36.8万套，故障率最高 |
| **隔离开关** | GW5 | 35kV+ | 常见于变电站 |
| **开关柜** | KYN28（铠装式） | 10kV | 最常见的金属封闭开关柜 |
| **环网柜** | XGN15 | 10kV | 配电网环网柜 |
| **负荷开关** | 负荷开关-熔断器组合 | 10kV | 630kVA以下变压器高压侧 |

每种型号的**操作规程、检修要点、安全注意事项**都不同，运检人员靠大脑记忆或翻纸质手册，效率低、容易出错。

### 1.2 核心痛点

| 痛点 | 描述 |
|------|------|
| **型号多，记不住** | 数十种开关型号，每种操作规程不同，新员工上手慢 |
| **现场查资料难** | 穿绝缘手套、戴安全帽，无法翻手册或操作手机 |
| **故障排查靠经验** | 报警信息复杂，现场难以快速定位问题，依赖老师傅 |
| **远程支援困难** | 电话描述设备状况不直观，专家无法"看到"现场 |
| **操作票执行风险** | 倒闸操作步骤严格，跳步或错步可能导致严重事故 |
| **巡检记录费时** | 手写记录再录入系统，重复劳动，数据易丢失 |
| **新员工培训周期长** | 需跟师傅学习1-2年才能独立作业 |

## 二、AR眼镜解决方案设计

### 2.1 整体架构

```
┌─────────────────────────────────────────────────┐
│                  云端平台                         │
│  ┌──────────┐ ┌──────────┐ ┌──────────────────┐ │
│  │ 设备知识库 │ │ AI识别引擎│ │ 远程协作平台     │ │
│  │(全型号资料)│ │(图像识别) │ │(专家实时连线)    │ │
│  └──────────┘ └──────────┘ └──────────────────┘ │
└───────────────────┬─────────────────────────────┘
                    │ 5G/WiFi
┌───────────────────┴─────────────────────────────┐
│              AR智能眼镜（现场端）                   │
│                                                   │
│  摄像头扫描 → AI识别开关型号 → 叠加显示操作指导      │
│  语音交互 → 免手动操作 → 自动生成巡检报告           │
│  第一视角直播 → 远程专家标注指导 → 实时协作          │
└─────────────────────────────────────────────────┘
```

### 2.2 五大核心功能

#### 功能一：开关设备智能识别

**解决痛点：型号太多记不住**

- 摄像头对准开关设备 → AI自动识别型号（ZN63? GW4? KYN28?）
- 识别后立即在视野中叠加显示：
  - 设备铭牌信息（型号、额定电压、额定电流、出厂日期）
  - 当前运行状态（分闸/合闸、带电/停电）
  - 上次检修时间、累计操作次数
  - 关联保护装置信息

**实现方式：**
- 预训练设备图像识别模型，覆盖青浦区所有在运开关型号
- 辅助手段：识别设备铭牌二维码/RFID标签
- 离线识别（无网络时也可用本地模型）

#### 功能二：操作票AR引导

**解决痛点：倒闸操作步骤多、易出错**

- 操作票步骤逐条在视野中高亮显示
- 每一步操作前，AR标注要操作的具体设备位置
- 操作确认后自动勾选、进入下一步
- 防误功能：如果操作顺序错误，立即语音+视觉告警

**典型场景：**
```
第3步：断开 #2主变10kV侧 ZN63断路器
  → AR在视野中用绿色箭头指向该断路器
  → 叠加显示："请确认断路器在分闸位置"
  → 语音播报："第三步，断开二号主变十千伏侧真空断路器"
  → 操作完成后自动拍照存档
```

#### 功能三：远程专家协作

**解决痛点：现场遇到疑难问题无法及时解决**

- 一键连线后方专家（运检部技术骨干/厂家工程师）
- 专家看到现场第一视角实时画面
- 专家可在画面上**标注、圈点、画箭头**，标注会实时显示在现场人员视野中
- 支持插入3D模型、技术图纸叠加显示

**效果参考：**
- 国网杭州：平均故障处理时间缩短60%
- 南方电网：抢修时间缩短50%

#### 功能四：智能巡检工作流

**解决痛点：巡检记录费时、数据易丢失**

- AR自动规划巡检路线，逐设备引导
- 到达设备后自动弹出巡检项目清单
- 语音录入检查结果："外观正常"、"表计读数35.2kV"
- 拍照自动关联设备档案
- 异常自动标红、生成缺陷工单
- 巡检完成自动生成报告，同步至PMS系统

#### 功能五：培训与知识库

**解决痛点：新员工培训周期长**

- 对准任意设备，调出3D拆解动画和工作原理讲解
- 模拟操作训练模式（AR叠加虚拟操作步骤）
- 常见故障案例库：展示历史故障现象、原因、处理方法
- 老师傅的检修经验录制成AR教程，知识沉淀传承

## 四、实施路线图

### 第一阶段：试点验证（1-3个月）

1. **设备资料数字化**
   - 梳理青浦区所有开关设备型号清单
   - 建立设备知识库：每种型号的操作规程、检修要点、常见故障
   - 拍摄设备照片训练AI识别模型

2. **硬件部署**
   - 采购3-5台X-Craft试用
   - 选择1-2个典型变电站（如设备型号较多的110kV站）

3. **基础功能验证**
   - 设备识别准确率 ≥ 95%
   - 远程协作连通性测试
   - 巡检工作流完整跑通

### 第二阶段：深化应用（4-6个月）

4. **操作票AR化**
   - 将常用操作票转为AR引导格式
   - 防误闭锁逻辑集成

5. **与现有系统对接**
   - PMS（生产管理系统）数据互通
   - 缺陷管理系统自动推送
   - 巡检报告自动归档

6. **培训体系建立**
   - 新员工AR培训教程
   - 老师傅经验AR化录制

### 第三阶段：全面推广（7-12个月）

7. **扩展到全区运检班组**
8. **AI模型持续优化**（识别率 → 99%+）
9. **效果评估与优化**
